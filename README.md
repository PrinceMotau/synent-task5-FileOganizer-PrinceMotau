# synent-task5-FileOrganizer-Prince

A Python script file organizer application that sorts and moves files into respective folders automatically.

# Feautures
- Script scans specified directory (Downloads)
- Cheks files extensions.
- Files are grouped arcording to extension type.
- Files are moved into respective folders(Images, Videos, Docs) based on file type automatically.

# Tech stack
- Python 3

# How to run
- Copy repository.
- Go to project folder.
- Run the script
